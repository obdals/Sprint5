Application Overview

CampusURP is an interactive and centralized platform designed for university students to easily access essential campus services. Users can book study rooms, explore department information, view upcoming campus events, and quickly get in touch with the administration â€” all from one user-friendly portal.

Technology Overview

  • Languages: HTML, CSS, JavaScript
  • Libraries: Custom Vanilla JS, optional CSS animations
  • Platform: Static web application
  • Hosting: Azure Static Web Apps

Proposal Last Updated

April 4, 2025

Feature List (Prioritized)

  • Interactive Study Room Booking (Individual & Group)
  • Dynamic Department Directory with Pop-up Info
  • Campus Events Display
  • Contact Us Form with Confirmation
  • Scroll-to-top navigation and responsive layout

Future Feature List

  • User Login System (Authentication)
  • Admin Panel for Event & Room Management
  • API Integration for Live Booking & Events
  • Room Availability Calendar View
  • Email Notifications upon Booking

Basic Technical Features

  • Dynamic form handling (Booking and Contact)
  • Unique ID generation for bookings and messages
  • Loader animations for better UX
  • Responsive layout and mobile-friendly design

Advanced Technical Features

  • Pop-up confirmations for booking and form submissions
  • Dynamic content rendering (Department Info)
  • Preparations for API integration and scalability
  • Custom styling and transitions for interactive UI/UX
